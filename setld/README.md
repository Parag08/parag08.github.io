# Settld Mobile Demo

A simple browser-based mobile app demo for Settld, a relocation concierge product that pairs newcomers with a trusted local buddy.

## Run

Open `index.html` in a browser, or serve the folder locally:

```bash
python -m http.server 5177 -b 127.0.0.1
```

Then visit:

```text
http://127.0.0.1:5177/
```

## Demo Flow

- Choose a destination city.
- Review matched local concierges.
- Tap a concierge to open the subscription screen, or choose `Schedule first`.
- Select 3, 7, or 30 days of settling support.
- Keep airport pickup and care package selected.
- Subscribe and see the confirmation toast.
